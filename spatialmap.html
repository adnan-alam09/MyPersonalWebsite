Spatial Map System

My First Embedded Systems Project

I programed a microcontroller using C firmware to control a steppor motor to obtain 360 degree of spatial data. I used UART and I2C communication protocol to transfer this data into the PC where the data was mapped using Pyhton's Open 3D Visualization Package. Through this project, I was able to demonstrate my electrical circuity knowledge and combine it with my Software skills!
